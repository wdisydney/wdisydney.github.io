Meet your freshest full-stack developers.

We are General Assembly Sydney's second-ever WDI class. We started with little to no coding experience and through 1000 hours over 12 weeks built a work-ready skills set and a number of our own projects.

Database Manipulation

We are trained with MySQL and PSQL databases and can apply the MVC framework to create applications.

Git and GitHub

We employed version-control and worked collaboratively - please check out our source code!

Command Line

We abandoned the Finder, memorising the commands programmers rely on every day.

Ruby

We learned beautiful, elegant Ruby syntax and were welcomed into its brilliant Sydney community.

Javascript

Our JS journey took us through all of the good parts including jQuery, Backbone and Underscore.

Ruby-on-Rails

We built custom apps integrating routing, controllers, views, databases and authentication.

HTML/CSS

We mastered the structural markup and visual language of the web including Haml and Sass.

Computer Science

We create lateral, logical solutions by applying fundamental CompSci concepts.

Responsive Design

We crafted our applications to provide optimal viewing experiences on any device.
